# Análisis Exploratorio de Datos

En esta sección se muestran resultados de EDA del set de datos de vancomicina. 


## Código

Se realizan mejoras en la forma de presentación de los datos de EDA respecto a documentación con tablas. 

## Farmacocinética

Los datos PK de vancomicina se utilizan con TAD como variable independiente, ya que no existe un buen registro del historial de dosificación. Por ejemplo, no hay registro en cada paciente de la hora de la dosificación para la ventana de muestreo.  Se utiliza el supuesto de estado estacionario (**SS**).
